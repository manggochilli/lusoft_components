# SourceCode Test

* 샘플 컴포넌트

<Button kind="primary" size="large" usrclass="c(green)">button test</Button>

* pre code 태그 적용

<pre><code><Button kind="primary" size="large" usrclass="c(green)">button test</Button></code></pre>

code 태그의 기본 스타일은 적용되지만 코드 내용을 보여주진 못하고 컴포넌트 내용을 그대로 실행함.

* 태그를 변환하여 표시

<code>
<div class="p(10) ">&lt;Button kind="<span class="c(red)">primary</span>" size="<span class="c(red)">lg</span>" usrclass="<span class="c(red)">c(green)</span>"&gt;button test&lt;/Button&gt;</div>
</code>

&lt;code&gt; 태그를 사용한 코드 보여주는 방법은 적용되지 않는것으로 추측됩니다.

< 를 변환하여 적용하면 코드를 보여줄 수는 있습니다.
